# Homebrew Toolify

Homebrew tap for [Toolify](https://github.com/mehmetsagir/toolify) - AI-powered voice transcription and translation tool.

## Installation

```bash
brew tap mehmetsagir/toolify
brew install --cask toolify
```

## Update

```bash
brew upgrade --cask toolify
```

## Uninstall

```bash
brew uninstall --cask toolify
brew untap mehmetsagir/toolify
```
